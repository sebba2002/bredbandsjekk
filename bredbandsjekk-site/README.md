# Bredbåndsjekk.no MVP

Next.js + Tailwind landing page for Bredbåndsjekk.no.

## Kjør lokalt

```bash
npm install
npm run dev
```

Åpne http://localhost:3000

## Deploy til Vercel

1. Last opp prosjektet til GitHub
2. Gå til Vercel.com
3. Importer GitHub-repoet
4. Deploy
5. Koble domenet bredbåndsjekk.no i Vercel

## Viktig

Skjemaet logger foreløpig lead-data i console. Neste steg er å koble `handleSubmit` til Google Sheets, Supabase eller CRM.
